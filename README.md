# Microservices Demo

Uses the following technologies:

- Docker (and Docker Compose)
- React
- Node.js
- MySQL
- Sequelize

## Setup

```sh
# in main directory
docker-compose up

# in a separate terminal, inside classifieds-app
yarn
yarn watch
```
